# Json Path Navigator Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.10] - 2026-01-06

### Fixed
- Fixed compatibility verification issues with newer IDE versions

### Changed
- Updated dependencies (Gradle IntelliJ Plugin, Changelog Plugin, JUnit)

## [1.0.9] - 2026-01-05

### Added
- Complete user documentation (Chinese and English)
- Improved search dialog UI

### Fixed
- Various bug fixes and performance improvements

## [1.0.8] - 2025-12-28

### Added
- Smart text selection from cursor position
- Auto-detect JSON path-like strings in current line

### Improved
- Better initial search text extraction

## [1.0.7] - 2025-12-20

### Added
- Index warm-up on project startup for faster first search
- Support for Word Index to improve search performance

## [1.0.6] - 2025-12-15

### Added
- Context menu integration (right-click to search selected text)
- Added to Edit menu

### Improved
- Dialog UI enhancements

## [1.0.5] - 2025-12-10

### Added
- Smart suggestions while typing
- Suggestion list shows matching key paths from project

## [1.0.4] - 2025-12-05

### Added
- Value preview in search results
- Support for displaying truncated long values

## [1.0.3] - 2025-12-01

### Added
- Line number display in results
- Click to navigate to exact line in file

## [1.0.2] - 2025-11-25

### Added
- Support for nested JSON object paths
- Path syntax using dot notation (e.g., `database.host`)

## [1.0.1] - 2025-11-20

### Fixed
- Compatibility issues with certain IDE versions
- Fixed search in large JSON files

## [1.0.0] - 2025-11-15

### Added
- Initial release
- Basic JSON key search functionality
- Keyboard shortcut support (`Ctrl+Alt+Shift+F` / `⌘+⇧+⌥+F`)
- Search dialog with results table
- Double-click to navigate to file location
- Support for all JetBrains IDEs (2024.1+)
